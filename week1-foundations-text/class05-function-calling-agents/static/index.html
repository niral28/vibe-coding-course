How the code works

① Function calling

The AI is told about two Python functions. Instead of doing math (which LLMs are bad at), it replies with "please run this tool with these arguments" — and our Python actually runs it.

add_expense(description, amount)
  → adds to the tab, returns total

calculate_split(num_people,
                tip_percent,
                tax_percent)
  → tax + tip, split per person

② The agentic loop

One message from you can take several API round-trips. The server keeps calling Gemini until it stops asking for tools and answers you — that's the loop in the console.

for step in range(6):
    response = gemini(messages, tools)
    if no tool_calls:
        show final answer; break
    run the tool
    send result back to Gemini

③ Country smarts

There's no country database here. The system prompt tells Gemini: if the user names a country, use your own knowledge of its tax rate and tipping culture, and pass those numbers to the tool.

"we're in Japan" →
  Gemini knows: 10% tax,
  tipping isn't customary
  → calculate_split(4, 0, 10)
