How It Works Under the Hood

The agent is equipped with a custom tool search_wikipedia(query). Instead of relying on its internal knowledge (which could be stale or hallucinated), the Python backend queries the live Wikipedia Search API, fetches the first matching article's introductory text, and feeds it directly into the Gemini context. 
pythondef search_wikipedia(query):
    # Query the Wikipedia Search API for matching titles
    search_response = requests.get(api_url, params={
        "action": "query", "list": "search", 
        "srsearch": query, "format": "json", "srlimit": 1
    })
    title = search_response.json()["query"]["search"][0]["title"]
    
    # Retrieve plain-text summary intro of that page
    page_response = requests.get(api_url, params={
        "action": "query", "prop": "extracts", 
        "exintro": True, "explaintext": True, 
        "titles": title, "format": "json"
    })
    content = page_response.json()["query"]["pages"][...]["extract"]
    
    return {"title": title, "url": f"https://wikipedia.org/wiki/{title}", "content": content}
The Anti-Hallucination System Prompt binds the LLM to extreme honesty. It explicitly orders the agent to act as a strict librarian: it cannot guess, it must run searches first, and it must cite the exact Wikipedia URL using the give_answer tool. 
text# The core instruction set governing agent integrity:
system_prompt = (
    "You are a careful and helpful research assistant.\n"
    "You must NOT use your own memory — facts may be outdated or wrong.\n"
    "You must call search_wikipedia to find facts FIRST.\n"
    "You must answer ONLY using what the search returned.\n"
    "If the search doesn't contain the answer, search again with different keywords, "
    "or say you couldn't find it (never make it up).\n"
    "When ready, call give_answer with the answer AND the source URL."
)
The program executes a multi-step Agentic Loop. If the model makes a search request, the backend interceptor runs the Wikipedia code, feeds the results back to the model, and calls Gemini again (up to 6 times). The loop stops immediately once Gemini invokes the give_answer tool. 
pythonfor step in range(6):
    response = client.chat.completions.create(
        model="gemini-3.5-flash", messages=messages, tools=tools
    )
    ai_message = response.choices[0].message
    
    # If AI has no more tool calls, exit loop
    if not ai_message.tool_calls:
        return ai_message.content

    # Run Wikipedia queries requested by model
    for call in ai_message.tool_calls:
        if call.function.name == "search_wikipedia":
            res = search_wikipedia(call.function.arguments["query"])
            messages.append({"role": "tool", "content": res})
            
        elif call.function.name == "give_answer":
            return call.function.arguments["answer"], call.function.arguments["source_url"]
