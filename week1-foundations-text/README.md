# Week 1: Foundations & Text Generation 📝

## Overview

Week 1 introduces the fundamentals of programming, AI concepts, and text generation. Students will set up their development environment, learn Python basics, explore the Gemini API, and build their first AI-powered text applications.

---

## Classes

### [Class 1: Course Intro & What is Generative AI?](class01-course-intro/)
- Overview of AI, generative models, and applications
- Understanding intelligence and benchmarks
- Hands-on: Set up Python, VS Code, and Git
- Run your first Python script

### [Class 2: Python Basics & Function Calling](class02-python-basics/)
- Variables, loops, functions in Python
- Introduction to function calling patterns
- Hands-on: Write a Python program using functions
- Process data with custom functions

### [Class 3: Text Generation with AI](class03-text-generation/)
- Introduction to prompt engineering
- Working with the Gemini API
- Building a chatbot with function calling
- Hands-on: Create an AI-powered Wordle game

### [Class 4: Project Workshop (Text)](class04-text-project-workshop/)
- Develop text-based mini-projects
- Implement chatbot with function calling
- Peer feedback and iteration
- Project refinement

---

## Learning Objectives

By the end of Week 1, students will:
- ✅ Understand the basics of generative AI and its applications
- ✅ Write basic Python programs with functions
- ✅ Make API calls to Google Gemini
- ✅ Implement function calling patterns
- ✅ Build a simple text-based AI application

---

## Key Concepts

- **Generative AI**: AI systems that create new content (text, images, audio, code)
- **Function Calling**: Allowing AI to invoke specific functions with parameters
- **Prompt Engineering**: Crafting effective prompts to get desired AI outputs
- **API Integration**: Connecting to external AI services via APIs

---

## Projects & Exercises

- Simple chatbot using Gemini API
- AI-powered Wordle game with validation
- Text processor with custom functions
- Story generator with function calls
