# Week 3: Multi-Modal Projects & Demos 🚀

## Overview

Week 3 is the culmination of the course! Students explore AI-assisted code generation, integrate multiple modalities (text, images, audio) in final projects, and present their work to peers, instructors, and industry professionals.

---

## Classes

### [Class 9: Code Generation & Multi-Modal AI](class09-code-generation/)
- Using AI for coding (e.g., GitHub Copilot, Claude Code)
- Peek under the hood: Training a tiny language model
- Combining text, images, and audio via MCP server
- Hands-on: Plan multi-modal final project with function calling

### [Class 10: Final Project Workshop](class10-final-project-workshop/)
- Develop final project with instructor guidance
- Integrate AI tools and MCP server
- Debug and optimize project workflows
- Team collaboration and peer support

### [Class 11: Guest Speaker Session](class11-guest-speaker/)
- Guest Demo: Industry/academic professional showcases AI project
- Q&A and advice for final demos
- Best practices for presenting technical work
- Career insights in AI field

### [Class 12: Final Project Demos & Wrap-Up](class12-final-demos/)
- **🎉 Demo Day!** Students present projects to audience
- Feedback from industry/academic professionals
- Course reflection and celebration
- Resources for continuing AI exploration
- Share projects online (GitHub, portfolio)

---

## Learning Objectives

By the end of Week 3, students will:
- ✅ Understand how AI assists in code generation
- ✅ Appreciate the fundamentals of language model training
- ✅ Build complete multi-modal AI applications
- ✅ Present technical projects effectively
- ✅ Have a portfolio piece to showcase their work

---

## Key Concepts

- **AI-Assisted Coding**: Using AI tools like Copilot, Claude, Cursor
- **Language Model Training**: Understanding GPT architecture basics
- **Transformer Architecture**: Attention mechanisms and neural networks
- **Multi-Modal Integration**: Combining text, image, audio, and code
- **Technical Presentation**: Communicating complex projects clearly

---

## Final Project Requirements

Students should demonstrate:
1. **AI Integration**: Use of Gemini API or other AI tools
2. **Function Calling**: Implementing tool use patterns
3. **User Interface**: Interactive element (web, CLI, or notebook)
4. **Project Theme**: Connection to one of the 5 course themes
5. **Demo-Ready**: 3-5 minute presentation with live demo

---

## Example Projects

- **AI Wellness Coach**: Mental health chatbot with audio responses
- **Screen Time Monitor**: App that suggests offline activities
- **College Assistant**: Study buddy with document analysis
- **Sustainability Advisor**: Eco-friendly lifestyle recommendations
- **Creative Writing Tool**: Story generator with image illustrations

---

## Demo Day Format

1. **Setup** (5 min): Prepare laptop and ensure demo works
2. **Presentation** (3-5 min):
   - Introduce the problem your project solves
   - Live demo of key features
   - Explain technical challenges and solutions
3. **Q&A** (2-3 min): Questions from audience
4. **Feedback**: Written feedback from professionals
