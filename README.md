# Vibe Coding: AI Remix 🎵🤖

## GSET 2025 AI Course

Welcome to **Vibe Coding: AI Remix** - an intensive, hands-on course introducing students to Generative AI through practical projects, live coding, and creative exploration!

> *"Letting AI help you code by vibe — fast, messy, creative, and fun."* - Andrej Karpathy

---

## 📚 Course Overview

- **Duration**: 3 weeks, 4 classes per week (12 total classes)
- **Target Audience**: Rising high school seniors (no prior CS knowledge required)
- **Focus**: Learn AI through building - text, images, audio, and code generation
- **Culmination**: Final project demos to peers and industry professionals

---

## 🗓️ Course Structure

### Week 1: Foundations & Text Generation
**Classes 1-4** | Python basics, Gemini API, function calling, text-based AI projects

### Week 2: Image & Audio Generation
**Classes 5-8** | Image generation, audio tools, MCP server setup, guest speaker

### Week 3: Multi-Modal Projects & Demos
**Classes 9-12** | Code generation, final projects, guest speaker, demo day

## 🚀 Getting Started

### Prerequisites

- Python 3.10 or higher
- Git
- VS Code (recommended)
- Google Gemini API key

### Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd vibe-coding-course
   ```

2. **Set up Python environment**
   ```bash
   # Using conda (recommended)
   conda create -n gemini-project python=3.11
   conda activate gemini-project

   # Or using venv
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```

3. **Install dependencies**
   ```bash
   pip install -r shared/requirements.txt
   ```

4. **Set up environment variables**
   ```bash
   cp shared/.env.example .env
   # Edit .env and add your API keys
   ```
