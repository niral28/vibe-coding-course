# Week 2: Image & Audio Generation 🎨🎵

## Overview

Week 2 expands into multi-modal AI, exploring image generation, audio synthesis, and MCP (Model Context Protocol) server basics. Students will create visual and audio content using AI tools and learn to integrate multiple modalities in their projects.

---

## Classes

### [Class 5: Image Generation with AI](class05-image-generation/)
- Introduction to image generation models (e.g., Stable Diffusion)
- Hands-on: Generate images from text prompts
- Integrate image generation with Python functions
- Create an AI art gallery

### [Class 6: Project Workshop (Images)](class06-image-project-workshop/)
- Build AI art gallery or visual storytelling projects
- Use functions to automate image generation workflows
- Peer feedback and refinement
- Enhance visuals with narrative

### [Class 7: Audio Generation & MCP Server Basics](class07-audio-mcp/)
- Overview of audio generation tools (e.g., AudioCraft, Gemini Live)
- Speech-to-text and text-to-speech integration
- Introduction to MCP server for AI task orchestration
- Hands-on: Generate audio clips and set up MCP server

### [Class 8: Guest Speaker Session](class08-guest-speaker/)
- Guest Demo: AI artist or developer shares their work
- Q&A and discussion on real-world AI applications
- Insights for project development
- Reflection on guest learnings

---

## Learning Objectives

By the end of Week 2, students will:
- ✅ Generate images from text prompts using AI models
- ✅ Create and manipulate audio with AI tools
- ✅ Understand speech-to-text and text-to-speech workflows
- ✅ Set up and use an MCP server for multi-modal tasks
- ✅ Build projects combining text, images, and audio

---

## Key Concepts

- **Image Generation**: Creating visuals from text descriptions
- **Stable Diffusion**: Popular AI model for image synthesis
- **Audio Synthesis**: Generating speech, music, and sound effects with AI
- **Speech-to-Text (STT)**: Converting audio to text
- **Text-to-Speech (TTS)**: Converting text to natural-sounding speech
- **MCP Server**: Protocol for managing AI model interactions

---

## Projects & Exercises

- AI art gallery with themed image generation
- Visual storytelling combining images and narration
- Voice-enabled chatbot with Gemini Live
- Multi-modal web agent with audio feedback

---

## Technologies

- **Image**: Stable Diffusion, DALL-E, Midjourney (concepts)
- **Audio**: Google Gemini Live, AudioCraft, Web Speech API
- **Integration**: Flask, WebSockets, MCP Server
